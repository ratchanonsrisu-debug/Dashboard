# Dashboard
ให้ทดลองสร้าง Dashboad อย่างง่าย เพี่อทดสอบความเข้าใจ โดยมี มี Graph อย่างน้อย 3 กราฟ ใช้ข้อมูลอะไรก็ได้ มี interactive ระหว่าง components เขียนโปรแกรมลงและเก็บข้อมูลใน Git ใช้หลัก Commit Early และ Commit Often ต้องมี Commit อย่างนั้น 25 Commit  ส่ง Git URL ใน Assignment นี้ อธิบายโค้ดอย่างเหมาะสม พร้อมทั้งอธิบายวิธีรันบน README.md
